SIMON GAME

INSTRUCTIONS TO FOLLOW:

  • The control of the Simon game unit.There should be four colored button,each corresponding to a different color like yellow,blue,red,green.

  • The unit wil produced different sound and light up the button in specfic pattern.

  • TO start the game PRESS A KEY TO BEGIN THE GAME.

  • The game will play in sequence of light and sounds by lighting up the button in a specific order.Watch and listen carefully to pattern

  • After the game display the sequence ,it will prompt you to repeat .Use the same colored button in the same order which you observed earlier.

  • If you correctly repeat the sequence the game will process next round will begin until you lose 😁

  • If you done any mistake and the repeat sequence incorrectlyThe game will produce background sound indicate and it will track your score.

  • This will become longer and more complex it help you testing your memory and concentration and also challenge yourself or with friends 😏.

  • After finish the game press any key to reset to start a new game.

  • HAVE FUN
